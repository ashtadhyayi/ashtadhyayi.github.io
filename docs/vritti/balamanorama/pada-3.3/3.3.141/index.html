BALAMANORAMA

3.3.141 वोताप्योः

वोताप्योः। अयमप्यधिकारः। वा आ उताप्योरिति छेदः। भूते इति, लिङ्निमित्ते लृङिति चानवर्तते। तदाह–उताप्योरित्यतः प्रागिति। उताप्योः समर्थयोरित्यतः प्रागित्यर्थः। नन्वनेन उताप्योरित्यतः प्राग्भूते लिङ्निमित्ते लृङ्वेत्याधिकाराक्रान्तत्वाद्भूतेचेति पूर्वमधिकारसूत्रं निर्विषयमित्यत आह– पूर्वसूत्रं त्विति। उताप्योः समर्थयोर्लिङित्यारभ्य इच्छार्थेभ्यो विभाषेत्यतः प्राक्भूते चे`ति पूर्वमधिकारसूत्रं प्रवर्तत इत्यर्थः। इमावधकारौ यत्रलिङ्विधिस्तत्रैव प्रवर्तेतते, नतु लडादिविधौ, लिङ्निमित्ताऽभावात्।
