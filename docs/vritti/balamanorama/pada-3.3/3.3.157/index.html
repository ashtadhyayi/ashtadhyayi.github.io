BALAMANORAMA

3.3.157 इच्छार्थेषु लिङ्लोटौ

इच्छार्थेषु लिङ्?लोटौ। इच्छार्थकधातुषु प्रयुज्यमानेषु लिङ्?लोटौ स्तः। सर्वलकारापवादः। असमानकर्तृकविषयमिदम्। समानकर्तृकेषु तु लिङ् चेति वक्ष्यते। इच्छामिति। भुञ्जीत भवानिति इच्छामीत्यन्वयः। कामप्रवेदने इति। परं प्रति स्वाभिप्रायाविष्करणे इच्छार्थेषु लिङ्लोटाविति विधिरित्यर्थः। इच्छन्करोतीति। परं प्रति स्वाभिप्रायाविष्करणाऽभावान्न लिङ्?लोटाविति भाव-। कामप्रवेदनेऽकच्चितीति सूत्रं तु प्रकरणादिना यत्र कामप्रवेदनं, नत्विच्छार्थकमुपपदमस्ति तद्विषयमिति बोध्यम्। इत उत्तरसूत्रं समानकर्तृकेषु तुमुन्निति तु कृदधिकारे व्याख्यास्यते।
