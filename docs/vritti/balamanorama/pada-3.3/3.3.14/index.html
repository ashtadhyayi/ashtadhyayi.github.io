3.3.14 लृटः सद्वा

लृटः सद्वा। लृटः शतृशानचौ वा स्त इत्यर्थः। व्यवस्थितेति। व्याख्यानादिति भावः। नित्यमिति। तेन तिङां निवृत्तिः। अप्रथमासामानाधिकरण्ये उदाहरति– करिष्यन्तमिति। प्रत्ययये परत उदाहरति– कारिष्यत इति। उत्तपदे उदाहरति– करिष्यद्भक्तिरिति। करिष्यन्ती भक्तिरिति। कर्मधारयः। सम्बोधने उदाहरति– हे करिष्यन्निति। शयिष्यमाणाभोक्ष्यन्ते यवना इति लक्षणे उदाहार्यम्। हेतावुदाहरति– अर्जयिष्यन्वसतीति। प्रथमासामानाधिकरण्येऽपि क्वचिदिति। अप्रथमासामानादिकरण्याऽभावेऽपि क्वचिदित्यर्थः। कदाचिदित्यपि द्रष्टव्यम्। इदं च लृट् शेषे चेति भविष्यदधिकारविहिते लृट�एव प्रवर्तते इति अनवक्लृप्त्यमर्षेत्यत्र भाष्ये स्पष्टम्।
