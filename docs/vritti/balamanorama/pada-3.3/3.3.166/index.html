BALAMANORAMA

3.3.166 अधीष्टे च

अधीष्टे च। लोट् स्यादिति। अधीष्टे विहितस्य लिङोऽपवादः। अधीष्टं सत्कारपूर्वको व्यापार इत्युक्तम्। त्वं स्माध्यापयेति गुरुं प्रत्युक्तिः। स्मे लोडधीष्टेचेत्येकसूत्रत्वेन सिद्धए योगविभागस्तु ऊर्ध्वमौहूर्तिक इत्यनुवृत्तिनिवृत्त्यर्थः। कालसमयेलासु तुमुन्नित्युत्तरसूत्रं तु कृदधिकारे व्याख्यास्यते।
