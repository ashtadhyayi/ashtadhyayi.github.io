BALAMANORAMA

3.3.142 गर्हायां लडपिजात्वोः

गर्हायाम्। अपि जातु अनयोर्द्वन्द्वः। अत्र वोताप्योरिति भूते लिङ्निमित्ते लृङ् वा इत्यधिकारो न संबध्यते। लड्विधानेन तद्विषये लिङ्?निमित्ताऽभावात्। कालत्रये इति। वर्तमाने भूते। भविष्यति चेत्यर्थः। भविष्यतीति निवृत्तम्। अतः कालसामान्ये लडिति भावः। परत्वादिति। अनवकाशत्वाच्चेत्यपि द्रष्टव्यम्।अपि जायामित्यत्र ,जातु गणिका`मित्यत्र अपि- जातुशब्दौ निन्दाद्योतकौ। तदाह– गर्हितमेतदिति।

परिचयः प्रतिस्पन्दः .
