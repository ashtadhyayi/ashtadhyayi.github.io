BALAMANORAMA

3.3.134 आशंसावने लिङ्

#आशंसावचने लिङ्। आशंसायाः प्राप्तीच्छाया भूते असंभवाद्भविष्यतीति लभ्यत इति मत्वाह– भविष्यतीति। आशंसायां भूतवच्चेत्यस्यापवादः। तदाह– नतु भूतवदिति। गुरुश्चेदिति। गुरुरुपेयाच्चेत् क्षिप्रमधीयीयेत्याशंसे इत्यन्वयः। क्षिप्रयोगेऽपि परत्वाल्लिङेव, नतु लृडिति भावः।
