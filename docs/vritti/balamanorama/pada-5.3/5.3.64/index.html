BALAMANORAMA

5.3.64 युवाऽल्पयोः कनन्यतरस्याम्

युवाल्पयोः। इष्ठेयसोरिति। अजादी इत्यनुवृत्तस्य सप्तम्या विपरिणामादिति भावः। कनिष्ठः कनीयानिति। अयमनयोरतिशयेन युवा अल्पो वेत्यर्थः। पक्षे यविष्ठ इति। युवन्शब्दादिष्ठनि स्थूलदूरेति वनो लोपे उकारस्य गुणेऽवादेशे रूपम्। अल्पिष्ठ इति। अल्पशब्दादिष्ठनि टिलोपः। इत्यादीति। यवीयान्, अल्पीयानिति रूपद्वयमादिपदग्राह्यम्।
