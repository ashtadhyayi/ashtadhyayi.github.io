VRITTI

3.3.170 आवश्यकाधमण्र्ययोर्णिनिः।

आवश्यकम्िति। मनोज्ञादित्वाद्()वुञ्।उपाधिरयम्िति। प्रत्ययार्थविशेषणमित्यर्थः। नोपपपदम् इति। उपपदत्वे तु कारी, हारीति रूपं नस्यादित्यभिप्रायः।ओरावश्यके(3.2.125) इत्यत्र प्रतिपादितमेवावश्यमुपाधिः, नोपपदमिति;तदिहापि तद्वदेवोपाधिकत्वमावश्यकस्य वदितव्यम्। तेन सह निर्द्देशादाधमण्र्यमप्युपाधिरेव, नोपपदमिति। यद्यावश्यकमपाधिर्नोपपदम्, उपपदमतिङ्(2.2.19) इति समासो न स्यात्, न चान्यल्लक्षणमस्ति येनेह समासः स्यादिति यो मन्येत्, तं प्रत्याह– मयूरव्यंसकादित्वात् समासः इति सहरुआं दायीइति।अकेनोर्भविष्यदाधमण्र्ययोः (2.3.70) इति षष्ठ()आं प्रतिषिद्धायां द्वितीयैव भवति॥
