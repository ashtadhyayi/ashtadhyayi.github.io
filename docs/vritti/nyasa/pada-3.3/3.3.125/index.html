NYASA

3.3.125 खनो घ च।

डो वक्तव्यःइत्यादि। सर्वत्र वक्तव्यशब्दस्य व्याख्येय इत्येषोऽर्थः। तत्रेदं व्याख्यानम्- चकारोऽत्र क्रियते, स चानुक्तसमुच्चयार्थः, तेन डादयोऽपि प्रत्यया भविष्यन्तीति। ननु च घञनुकर्षणार्थश्चकारः? नैतदस्ति; स्वरितत्वादेव हि घञपतिष्ठते। अथ किमर्थं खनेर्घः प्रत्ययो विधीयते, यावता कुत्वार्थं तस्य विधानम्, न च खनेः कश्चिदवयवः कुत्वभागस्ति? एवं तह्र्रेतज्ज्ञापयति–अन्येभ्योऽप्ययं भवतीति। तेन खलः,भगम्, पदमित्येवमादयः सिद्धा भवन्ति॥
