3.3.48 नौ वृ धान्ये।

अपोऽपवादःइति। ग्रहवृदृ (3.3.58) इत्यादिना प्राप्तस्य। नीवारा नाम व्रीहयो भवन्ति इति। एतेन यद्यपि धान्यशब्देन सामान्यवाचिनायं व्युत्पाद्यते, तथापि प्रत्ययान्तं शब्दशक्तिस्वाभाव्याद्धान्यविशेष ए वत्र्तत इति दर्शयति। उपसर्गस्यघञि (6.3.121) इति दीर्घत्वम्। निवरा इति। निव्रियत इति निवरा। स्वभावेनाबन्तमपि स्त्रियां वत्र्तते। एवञ्च यद् घञबन्तो घाजन्तश्चेति लिङ्गाधिकारेऽबन्तस्य पुंल्लिङ्गताप्रतिपादनं तत् प्रायिकं वेदितव्यम्॥
