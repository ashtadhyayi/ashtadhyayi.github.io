VRITTI

3.3.118 पुंसि संज्ञायां घः प्रायेण।

समुदायेण चेत् इत्यादि। प्रकृतिप्रत्ययसमुदायेन यदि संज्ञा गम्यत एवं प्रत्ययो भवति, नान्यथा। अकार्त्स्न्यार्थम् इति। अकार्त्स्न्यम् = असाकल्यम्, प्रयोजनमर्थो यस्य तत् तथोक्तम्। एतेनाकृत्स्नविषयतां प्रत्ययस्य दर्शयति। दन्तच्छदः इति। छद अपवारणे (धा।पा।१९३५), चुरादिणिच्। छादेर्घेऽद्वयुपसर्गस्य (6.4.96) इति ह्यस्वः, पूर्ववत् समासः। प्रसाधनम्िति।राध साध संसिद्धौ(धा।पा।१२६२), १२६३), अस्माद्धेतुमण्ण्यन्तात् पूर्वसूत्रेण करणे ल्युट्,कुगतिप्रादयः (2.2.18) इति समासः॥
