3.3.94 NYASA सर्ववृत्तिसहितमीक्षताम्

3.3.94 स्त्रियां क्तिन्।

भावादौ इति। आदिशब्देनाकर्त्तुः कारकस्योपादानम्। घञजपामपवादः इति। य इकारान्तास्तेभ्योऽचोऽपवादः, य ऋकारान्ता उवर्णान्ताश्च, तेभ्योऽपः, शेषेभ्यो घञः। वक्तव्य इति। व्याख्येय इत्यर्थः। व्याख्यानं तु कृत्यल्युटो बहुलम् (3.3.113) इत्येतदाश्रित्य कत्र्तव्यम्। आप्तिः इत्यादि। आप्लृ व्याप्तौ (धा।पा।१२६०), राध साध संसिद्धौ (धा।पा।१२६२), दीपी दीप्तौ (धा।पा।११५०), रुआन्सु ध्वन्सु अवरुआंसने (धा।पा। ७५४,७५५) डुलभष् प्राप्तौ (धा।पा।९७५)। लभेति षित्त्वात् षिद्भिदादिभ्योऽङ (3.3.104) इत्यङि प्राप्ते शेषेभ्यस्तु गुरुत्वात् गुरोश्च हलः(3.3.103) इत्यकारे क्तिन विधीयते। इष्टिः इति। वच्यादिसूत्रेण (6.1.15) सम्प्रसारणम्।कीर्णिः,गीर्णिः, शीर्णिः इति। ऋत इद्धातोः (7.1.100) इतीत्त्वम्, रषादिसूत्रेण (8.4.1) णत्वम्। यथा रदाभ्याम् (8.2.42) इत्यादिना कीर्ण इति निष्टायास्तकारस्य नत्वं भवति, तथेहापि क्तिनः। यथा ल्वादिभ्यः(8.2.44) इति लून इत्यत्र निष्ठातकारस्य नत्वं तथा लूनिः, यूनिःइत्यात्रापि क्तिनः॥
