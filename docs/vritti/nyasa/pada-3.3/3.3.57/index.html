3.3.57 ऋदोरप्। सर्ववृत्तिसहितमीक्षताम्

पित्करणं स्वरार्थम् इति। अनुदात्तौ सुप्पितौ (3.1.4) इत्यनुदात्तत्वं यथा स्यात्। ऋदोरप् इत्यत्र ऋकारस्तपरः स्यात्। एवं सति तादपि परस्तपरः इत्युकारोऽपि तस्य परः स्यात्। ततश्च तपरस्तत्कालस्य (1.1.69) इति परिभाषोपस्थानादिहैव स्यात्– यवः,स्तव इति; लवः,पव इत्यत्र तु न स्यात्। दकारे तु अण् सवर्णान् गृह्णाति--भुवः गृह्णाति(व्या।पा।६६?) इति दीर्घादपि भवति। तस्मादुकारोऽयं सवर्णं गृह्णाति। अत्र च लिङ्गम्--भुवः प्रभवः(1.4.31) इति निर्देशः।तपरे तु भवतेरप् न स्यात्, ततश्चप्रभवःइति निर्देशो नोपपद्यते। तस्य तु दकारस्य मुखसुखार्थत्वादन्यत् प्रयोजनं न सम्भवतीत्येतत्सर्वमालोच्याह--दकारो मुखसुखार्थः` इति॥
