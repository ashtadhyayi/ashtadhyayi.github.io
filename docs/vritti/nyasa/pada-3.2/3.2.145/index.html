3.2.145 प्रे लपसृद्रुमथवदवसः।

प्रलापी, प्रसारी इति। लप व्यक्तायां वाचि(धा।पा।४०२), सृ गतौ (९३५)। प्रद्रावी, प्रमाथीइति। दु द्रु गतौ (धा।पा।९४४ ९४५), {मथे विलोडने-धा।पा।} मध विलोडने (धा।पा।८४८)। नाच्छादनार्थस्य इति। न वस आच्छादने (धा।पा।१०२३)इत्येतस्य॥
