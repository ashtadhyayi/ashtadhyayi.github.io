5.3.7 पञ्चम्यास्तसिल्?।

(7.3.103) इत्येत्वम्()। कुत्रेति,अच्च घेः(7.3.118) इत्यौत्त्वम्()। ततः, तत्रेति,ङसिङ्योः स्मात्स्मिनौ(7.1.15) इति स्मात्स्मिनौ। तस्मान्नादेशा अमी क्रियन्ते। यद्येवम्(),तसु उपक्षये(धा। पा। १२१२) इत्यस्मात्? तस्यन्नित्यत्र धातुरूपस्य पञ्चम्याः परस्यतसेश्च(५।३।८) इति तसिलादेशः प्राप्नोति? नैष दोषः; स्वार्थिकविज्ञानान्न भविष्यति। स्वार्थिका ह्रेते तसिलादयः, तन्मध्ये तसिलादेशो विधीयमानस्तथा विज्ञायते यथा स्वार्थे क्रियामाणः, स्वार्थिकत्वं नातिक्रामति।अपादाने चाहीयरूहोः` (5.4.45) इत्यनेन विहितस्तस्य स्व#आर्थिकस्य साथाने भवंस्तसिलपि स्वार्थिक एव भवति। धातुस्तु प्रातिपदिकादर्थान्तरे वत्र्तत इति तदादेशो भवंस्तसिल्? स्वार्थिको न भवति॥
