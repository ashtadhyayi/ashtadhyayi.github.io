पराशर्य। मण्डूकप्लुयाणिनिरिह सम्बध्यत इत्याह—णिनिः स्यादिति। तद्विषयताऽत्रेष्यते, तदर्थं छन्दोग्रहणमनुवर्त्त्यम्। सूत्रयोश्छन्दस्त्वं तु गौण्या वृत्त्या बोध्यम्। भिक्षुसूत्रमिति। चतुर्लक्षणीरूपम्। पाराशरिण इति। पाराशर्यो—व्यासः। अनन्तरापत्येऽपि गोत्रत्वेनोपचारात् गर्गादिभ्यःइति यञ्। आपत्यस्य चेति यलोपः। अध्येत्रणस्तु प्रोक्तादिति लुक्। भिक्षुनटसूत्रयोः किम्?। पराशरम्। शैलालम्।

रथाद्यत्। अणोऽपवादः। रथाद्रथाङ्गेइति वार्तिकमिभिप्रेत्याह—रथ्यं चक्रमिति। रथसीताहलेभ्यो यद्विधौइति तदन्तविधिरूपसङ्ख्यायते। परमरथ्यम्। उत्तमरथ्यम्। द्वयो रथयोरङ्गं द्विरथम्। इह द्विगोर्लुगनपत्येइति यतो लुक्। ननु यत्प्रत्ययोऽत्र रथाङ्ग एव यदीष्यते कथं तर्हि रथस्य वोढा रथ्यःइति ?। तद्वहतीत्यनेनेति चेत्। एवं तर्हि अयमेव यद्रथाङ्ग इव वोढर्यीष्यतां, तद्वहति रथयुगप्रासङ्गमित्यत्र रथग्रहणं त्यज्यतामिति चेत्।मैवम्। द्वौ रथौ वहति द्विरथ्य इत्यत्र द्विगोर्लुगनपत्येइति प्राग्दीव्यतीयस्य लुक्?प्रसङ्गात्। तद्वहतीति यत्प्रत्ययस्य तु प्राग्दीव्यतीयत्वाऽभावान्न लुगिति द्विरथ्यइति सिध्यतीति। ननु द्विगोर्लुगनपत्येइत्यत्राऽचीत्य[स्या]पकर्षणात्कथमत्र यतो लुक्प्रसक्तिरिति चेत्। अत्राहु—तद्वहतीत्यत्र रथग्रहणमेव ज्ञापकम्—एतस्य यतो हलादेरपि लुग् भवतीति। अन्यथा तत्र रथग्रहणं व्यर्थमेव स्यात्। तथा च द्वयो रथयोरङ्गमिति विग्रहे द्रिरथमिति प्रयोगः सुस्थ इति।
