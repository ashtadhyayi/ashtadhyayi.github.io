परिभाषेयमादेशनियमार्था।स्थाने प्राप्यमाणानाम् इति। एतेन नेदं विधायकं वाक्यम्, किं तर्हि? लक्षणान्तरेण प्राप्तानामनन्तरतमानामितरेषां च योऽन्तरतमः, स एव भवतीति नियमार्थमिति दर्शयति। विधायकत्वे हि दधि मधु इतादावपि प्रसज्यते। किं पुनरत्रानिष्टम्? यावतान्तरतम्याद् दधिमधुशब्दयोस्तावेवादेशौ भविष्यतः, तयोरप्यादेशयोरन्यौ दधिमधुशब्दौ यावादेशौ स्याताम्, तयोरप्यन्यावित्येवमपरिसमाप्तादेशपर- म्पराविधानादनवस्था स्यात्। ततश्चार्थप्रत्यायनं प्रति शब्दानां विनियोगो न स्यात्न्वाख्यायकस्य शास्त्रस्य प्रवृत्त्यनुपरमात्। शास्त्रोपरतौ ह्रन्वाख्यातास्ते प्रयुज्यते, नान्यथेत्यनिष्टम्। ननु च सकृत्प्रवृत्तौ लक्षणस्य चरितार्तत्वात् पुनः प्रवृत्तिर्न भविष्यति, नैतदस्तिच यत्र हि प्रयोजनार्था प्रवृत्तिस्तत्रैतदुक्तम्, इह पुनरकस्मादन्तरतमो विधीयत इत्यनुपरतैवास्य प्रवृत्तिः। नियमाकत्वेऽप्यस्य यदीदं विहितानां नियमं कुर्यात् !
