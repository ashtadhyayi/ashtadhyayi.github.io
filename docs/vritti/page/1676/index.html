जुचङ्क्रम्य। क्रमिद्रमी यङन्तौ। क्रमु पादविक्षेपे, द्रम हम्म मीमृ गतौ। पूर्वेणेति। अनुदात्तेतश्चे त्यनेन। इह पदिग्रहणं सकर्मकार्थ मिति वृत्तिकृतोक्तम्। भाष्ये तु अनभिधानादेव सकर्मकान्न भविष्यतीत्युक्तत्वात्। ज्ञापनार्थमेव तदित्याशयेनाह— बाधा माभूदित्यादि। परस्परमिति। निन्दादिसूत्रे हि तच्चीलादिषु वासरूपविधिना तृजादयो नेति ज्ञापितम्, इह तु ताच्छीलिकेषु परस्परं नेति ज्ञापितमिति विवेक इति भावः।
