कालेभ्यो भववत्। कालवाचिभ्यो भवेऽर्थे येन विशेषणेन ये प्रत्यया वक्षयन्ते, ते साऽस्य देवतेत्यर्थे कालवाचिभ्यस्तेनैव विशेषणेन भवन्तीत्यर्थः। मासिकमिति। मासो देवता अस्येति विग्रहः। कालाट्ठञ्। प्रावृषेण्यमिति। प्रावृट् देवता अस्येति विग्रहः। प्रावृष एण्यः।
Vritti

महाराजप्रोष्ठपदाट्ठञ्। माहाराजिकमिति। महाराजो वैश्रवणः, स देवता अस्येति विग्रहः। प्रौष्ठपदिकमिति। प्रोष्ठपदो देवता अस्येति विग्रहः।

भिक्षादिभ्योऽण्। तस्य समूह इत्येव। भैक्षमिति। अत्र अचित्तहस्तीति वक्ष्यमाणठगपवादोऽण्। गार्भिणमिति। गर्भशब्दान्मत्वर्थीये इन्प्रत्यये कृते प्रत्ययः परश्च आद्युदात्तश्चेति इकारस्य उदात्तत्वे अनुदात्तं पदमेकवर्जमिति शिष्टस्यानुदात्तत्वे गर्भिन्शब्दः अनुदात्तादिः। ततो नान्तलक्षणङीपि तस्य अनुदात्तो सुप्पितौ इत्यनुदात्तत्वे गर्भिणीशब्दोऽप्यनुदात्तादिरेव। ततः समूहेऽर्थे अनुदात्तादेरञिति वक्ष्यमाणे अञि प्राप्ते भिक्षादित्वादणिति भावः। अणि प्रत्ययस्वरेणान्तोदात्तत्वम्, अञि तु ञ्नित्यादिर्नित्यम् इत्याद्युदात्तत्वमिति स्वरे विशेषः। अणि टिलोपाऽभावोऽपि प्रयोजनमिति दर्शयति–इह भस्येति। गर्भिणीशब्दादणि सति भस्याऽढे इति पुंवत्त्वेन ङीपो निवृत्तौ गर्भिन् अ इति स्थिते नस्तद्धिते इति टिलोपे प्राप्ते सतीत्यर्थः।

गोत्रोक्षोष्ट्रो। एभ्य इति। गोत्र, उक्षन्, उष्ट्र, उरभ्र, राजन्, राजन्य, राजपुत्र, वत्स, मनुष्य, अज-एतेब्य इत्यर्थः लौकिकमिह गोत्रमिति। नतु पारिभाषिकमित्यर्थः। अत्र लौकिकं गोत्रं किमित्यत आह–तच्चापत्यमात्रमिति। प्रवराध्याये परिगणितं पुत्रपौत्रादि कृत्स्नापत्यमित्यर्थः। अपत्याधिकारादन्यत्र लौकिकमेव गोत्रं गृह्यते, नतु पारिभाषिकं गोत्रमिष्यते इति स्त्रीपुंसाभ्यामित्यादिसूत्रबाष्ये सिद्धान्तितत्वादिति भावः।
