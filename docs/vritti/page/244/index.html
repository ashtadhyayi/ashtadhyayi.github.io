तदाह–अधातोरुगित इत्यादिना। नुमि मकार इत्। उकार उच्चारणार्थः। मित्त्वादन्त्यादचः परः। अजिति अच्प्रत्याहारो न गृह्यते, व्याख्यानात्, नपुंसकस्य झलचः इत्यज्ग्रहणाच्च। अन्यथा उगिदचामित्येव सिद्धे
