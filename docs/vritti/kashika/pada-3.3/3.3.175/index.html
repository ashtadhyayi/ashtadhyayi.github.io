KASHIKA

3.3.175 माङि लुङ्

माङि उपपदे धातोः लुङ् प्रत्ययो भवति। सर्वलकाराणाम् अपवादः। मा कार्षीत्। मा हार्षीत्। कथम् मा भवतु तस्य पापम्, मा भविष्यति इति? असाधुरेव अयम्। केचिदाहुः, अङिदपरो माशब्दो विद्यते, तस्य अयं प्रयोगः।
