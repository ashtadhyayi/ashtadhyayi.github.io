7.3.83 जुसि च

जुसि च। चकारः स्पष्टप्रतिपत्त्यर्थः। क्सस्याचीत्यतोऽत्यतोऽचीत्यनुवर्तत इत्याह– अजादाविति। जागृयुरिति। एवं श्रृणुयुः चिनुयुरित्यादावपि गुणो नेति बोध्यम्। यासुटो ङित्तवाच्छृणुयुरित्यादौ गुणो नेति प्राचोक्तिस्तु मनोरमायां दूषिता, जुसि चेति गुणस्य निषेधापवादत्वादिति। केचित्तु–जुसि चेत्यत्र उसीत्यावर्त्त्य उरुआऊपे जुसीति व्याख्यानान्नोक्तदोष इत्याहुः। अजादौ जुसीति समाधानं तु भाष्यारूढम्। जागर्तेः सिच इटि कृते यणादिप्राप्तिक्रमं दर्शयति– जागृ- इसित्यत्रेति।तत्र कृते इति। हलन्तलक्षणाया वृद्धेर्जागर्तिगुणेन बाधः, तेन गुणेनहलन्तत्वसंपादनात्। या हि गुणप्रवृत्तिसमये वृद्धिः सा बाध्यते नान्येति भावः। अजागरिष्टाम्। अजागरिषुः। अजागरीदित्यादि। दरिद्रा।
