3.1.32 सनाद्यन्ता धातवः

सनाद्यन्ताः। सदनादय इति। सन्?क्यच्?काम्यचज्?क्यङ्?क्यषोऽथाचारक्विप्?णिज्?यङौ तथा। यगाय णिङ् चेति द्वादशीमी सनादयः। धातुत्वादिति। प्रत्ययविशिष्टस्य धातुत्वादित्यर्थः। सुप्तिङन्तमित्यनेन संज्ञाविधौ प्रत्ययग्रहणे तदन्तग्रहणं नास्तीतिज्ञापितत्वाद्विशिष्टस्य धातुसंज्ञालाभाय अस्मिन् सूत्रे अन्तग्रहणं कृतम्। भूवादयो धातवः इत्यस्यानन्तरं सनाद्यन्ताश्चेति न सूत्रितं, सनादयो द्वादशेवेति निर्धारणाऽलाभापत्तेः। निर्धारणाऽलाभापत्तेः। सनाद्यन्ता धातवः इत्यस्यानन्तरं तु भूवादयंश्चेति पठित्वा धातव इत्येतत्तन्न त्यक्तुं शक्यम्।
