3.1.134 नन्दिग्रहिपचादिभ्यो ल्युणिन्यचः

नन्दिग्रहि। द्वन्द्वान्ते श्रूयमाण आदिशब्दः प्रत्येकं संबध्यते। तदाह– नन्द्यादेर्ल्युरित्यादि। मधुसूदन इति। मधुं दैत्यं सूदयतीति विग्रहः। षूद क्षरणे। इह अर्दिसूदिभ्यां कर्मण्यणि प्राप्ते नन्द्यादिपाठाल्ल्युः। विभाषण इति। भियो हेतुभये षुक्। ग्राहीति। णिनेर्णिदत्त्वादुपधावृद्धिः। स्थायीति। आतो युक्। मन्त्रीति। मत्रीति चुरादाविदित्पाठान्नुम्, णेरनिटीति णिलोपः। विषयीति। षिञ् बन्धने। धात्वादेरिति षस्य सत्वे आदेशप्रत्यययो रिति प्राप्तस्य षत्वस्य सात्पदाद्योरिति निषेधादाह--इह षत्वमपीति।निपातनादित्यनुषज्यते।परिनिविभ्यःइति सूत्रेसितसयेति क्तान्ताऽजन्तसिनोतेर्ग्रहणाण्णिन्यन्तसिनोतेर्निपातनं विना षत्वं दुर्लभमिति भावः। जारभरेत्यादि। जारं बिभर्ति, �आआनं पचतीति विग्रहः। �आपाकोऽपीति। कर्मण्यणपि पक्षे भवतीति भावः। चेक्रिय इति। संयोगपूर्वत्वात्एरनेकाचःइति न यण्। लोलुव इति। इह सुबभवात्ओः सुपीति यणनेतिअचि श्नुधातुइत्युवङ्। मरीमृज इति।रीगृदुपधस्ये चे`ति रीगागमः।
