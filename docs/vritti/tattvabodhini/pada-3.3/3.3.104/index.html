3.3.104 षिद्भिदादिभ्योऽङ्।

षिद्भिदा। कथं तर्हि मुखाब्जगन्धलब्धे रिति माघः। प्रेक्षोपलब्धिरित्यमरश्च। षित्त्वादङि लभेत्येव ह्युचितम्। सत्यम्। अनर्थकास्तु प्रतिवर्णमनुपलब्धे रिति भाष्यप्रयोगाद्बाहुलकाद्वा क्तिन्नपि बोध्यः। विस्तरस्त्विह मनोरमायां बोध्यः। भिदेति। एवमादयोऽदन्ताः समुदाया एव गणे पठ�न्ते तत्र ये प्रकृतिभागा भिद-छिद इत्यादयस्ते आदिशब्देन सूत्रे निर्दिष्टाः। गणे विशिष्टपाठस्तु लोकप्रसिद्धार्थविशेषस्य कवचिदलाक्षणिककार्यस्य सङ्ग्रहार्थः। भित्तिरन्येति। भिद्यत इति भित्तिः = कुड�म्। छिदेति। द्वैधीकरण एवायम्। अन्यत्र तु छित्तिश्छिद्रम्। मृजेति। भृजू शुद्धौ, अस्याऽषित्त्वाद्भिदादौ पाठः।
