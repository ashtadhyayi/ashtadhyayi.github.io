3.3.139 लिङ्?निमित्ते लृङ् क्रियातिपत्तौ

हेतुहेतुद्भावादीति। अत्र केचित्– आदिशब्देनाशंसावचनं गृह्यत इति व्याख्याय आशंसावचने लिङो यन्निमित्तं तत्रापि क्रियातिपत्तौ भविष्यति लृङ्, गुरुश्चेदायास्यत्, आशंसे,- अहमध्येष्ये’ इत्याद्युदाहरन्ति। अन्ये तु भविष्यति मर्यादेत्यादिना भविष्यतीत्युपक्रम्य यो यो लिङ्ग विहितस्तन्निमित्त एव क्रियातिपत्तौ लृह् भवति, नान्यत्रेत्याहुः॥ लिङ्?निमित्तमिति। हेतुहेतुमतोर्लिङ् , इच्छार्थेष #उ लिङ्लोटौ इत्यादिलकारार्थप्रक्रियायां स्फुटीभविष्यति। अनिष्पत्ताविति। सुवृष्टिश्चेदभविष्यत्तदा सुभिक्षमभविष्यदिति तत्रैवोदाहरिष्यति।
