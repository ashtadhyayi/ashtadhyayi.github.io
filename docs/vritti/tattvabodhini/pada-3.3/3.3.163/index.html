3.3.163 प्रैषतिसर्गप्राप्तकालेषु कृत्याश्च

प्रैषतिसर्ग। प्रैषो विधिरिति। विधिग्रहणे कर्तव्ये प्रैषग्रहणं शिष्यबुद्धिवैशद्यार्थमित्याहुः। लोटोऽनुकर्षणमिति। लोट् चेति योगस्येदमेव प्रयोजनम्। एकयोगत्वे तु लिङोऽप्यनुकर्षणं स्यात्। प्राप्तकालार्थमिति। प्रैषतिसर्गयोः पूर्वेणैव सिद्धत्वादिति भावः।

परिचयः प्रतिस्पन्दः .
