3.3.137 कालविभागे चाऽनहोरात्राणाम्

कालविभागः = कालविशेषः। अनहोरात्राणामिति संबन्धषष्ठी। पूर्वसूत्रमिति। तदयमर्थः– मर्यादोक्तौ कालविशेषाद्यदवरं तस्मिन्प्रविभागे भविष्यत्यनद्यतनवन्न, स चेत्कालविभागोऽहोरात्रसंबन्धी न चेदित्यर्थः। इह कालविभागे सत्यवरस्मिन्ननद्यतनवन्नेत्युक्ते कस्मादित्यपेक्षायामर्थात्कालविशेषादिति लभ्यत इत्येके। केचित्तु कालविभाग इति पञ्चम्यर्थे सप्तमीत्याहुः। उत्तरार्थ इति। परस्मिन्विघाषेति सूत्रे कालमर्यादाया एवानुवृत्तिर्यथा स्यात्। भविष्यतीति किम् ?। यः संवत्सरोऽतीतस्तस्य यदवरं मासात्तत्र पयोऽपिबाम। मर्यादायामिति किम् ?। यो निरवधिः समयस्तस्य यदवरं मासात्तत्र पयः पातास्मः। पञ्चदशरात्र इति। नन्वत्र सङ्ख्यापूर्वं रात्रमिति क्लीब्तवेन भाव्यम्। न च पञ्चदश रात्रयो यस्मिन्पक्षे इति बहुव्रीह्याश्रयणात्पुंल्लिङ्गःसिद्ध इति वाच्यम्, `अहः सर्वैकदेशेट त्यच्प्रत्ययस्याऽभावप्रसङ्गादिति चेत्। सत्यम्। अतएव हि भाष्यप्रयोगात्क्लीबत्वं नेत्याहुः।
