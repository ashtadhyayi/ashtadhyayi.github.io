3.3.140 भूते च

भूते च। लिङ्?निमित्ते लृङ् स्याद्भूते भविष्यति च क्रियापत्तौ। अद्यतनाऽनद्यतनसाधारमं भूतसामान्यमिह ज्ञेयम्। अस्योदाहरणानि—उताप्योः समर्थयोर्लिङ् इत्यारभ्य समानकर्तृकेषु तुमुन्निति विहाय सप्तसूत्र्यां बोध्यानि, ततः प्राचीनेषु वोताप्योरित्यनेन विकल्पोक्तेः।
