3.3.131 वर्तमानसामीप्ये वर्तमानवद्वा

वर्तमान। स्वर्थे ष्यञिति। अस्मादेव सामीप्य इति निर्देशात्ष्यञि सिद्धे चतुर्वर्णाअदिभ्यः इति वार्तिकं प्रपञ्चार्थमित्याहुः। न चाऽत्र पूरणगुणेति निषेधात्षष्ठीसमासो नप्राप्नोतीति वाच्यम्, अनेनैव निर्देशेन तन्निषेधस्याऽनित्यत्वाऽभ्युपगमात्।तेन बुद्धमान्द्यमित्यादि सिद्धम्। येनोपाधिनेति। धातुविशेषाद्युपाधिना। तथाहि पूड�जोः शानन्विहितः, स ताभ्यामेव धातुभ्यां भवति,न तु धात्वन्तरात्। तच्छीलादिविशिष्टे कर्तरि अलङ्कृञित्यारभ्य भ्राजभासेति यावत् ये इष्णुजादय उक्तास्तेऽपि कृञादिभ्यो धातुभ्यस्तस्मिन्नुपाधौ सत्यैव भवन्ति। कदा आगतोऽसीति भूतकालेन प्रश्नवाक्यम्, आगच्छामीत्युदाहरणे भूतकालाभिव्यक्तयेऽयमिति प्रयोगस्त्विदानीमेवागममिति वर्तमानसमीपभूतकालद्योतनाय। तेन ह्यागमनाऽविनाभूतं यद्रूपं परिकरबन्धादियुक्तं तत्प्रतिनिर्दिश्यते। न च कदेति प्रश्ने उत्तरवाक्ययोः कालानुपादानादसङ्गतिः शङ्क्या, उत्तरत्राऽयमेषशब्दाभ्यामिदानीमिति कालावगमात्। सामीप्ये किम् ?। कदा आगतो भवान् ?। अस्मान्मासात्पूर्वस्मिन्मासे आगच्छम्। एतच्च सूत्रमाकरे प्रत्याख्यातम्। गङ्गासमीपे गङ्गात्वारोपवद्वर्तमानसमीपे वर्तमानत्वारोपसंभवात्।
