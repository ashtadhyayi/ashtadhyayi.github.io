3.2.134 आ केस्तच्छीलतद्धर्मतत्साधुकारिषु

आ क्वेः। शीलादीनां त्रयाणां विशेषणं समर्थयितुं निर्दिष्टैस्त्रिभिस्तच्छब्दैः प्रकृतिभूतधात्वर्थो निर्दिश्यते। क्विपमभिव्याप्तेति। भ्राजभासधुर्विद्युतोर्जीति वक्षमाण’मिति शेषः। कर्ता कटानीति। न लोकाव्ययेति षष्टीनिषेधः।
